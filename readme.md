# Grainy's Portfolio

A modern, responsive portfolio website showcasing AI/ML expertise, Data Science projects, and Full Stack development skills.

## 🚀 Features

- **Responsive Design**: Optimized for all devices using modern CSS.
- **Glassmorphism UI**: Sleek, transparent card designs with vibrant accents.
- **Dynamic Icons**: Integrated Lucide and Devicons for a professional look.
- **Project Showcase**: Detailed cards for AI, ML, and Web Development projects.
- **Training**: Highlighted section for specialized DSA training at CSE Pathshala.
- **Extracurricular Activities**: Showcasing hackathons and technical workshops.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+).
- **Icons**: [Lucide Icons](https://lucide.dev/), [Devicon](https://devicon.dev/).
- **Fonts**: Inter (via Google Fonts).

## 📁 Structure

- `index.html`: Main structure and content.
- `style.css`: All styling rules and design system.
- `script.js`: Interactive elements and icon initialization.
- `assets/`: 
  - `images/`: Project screenshots and certificates.
  - `resume/`: PDF version of the CV.
